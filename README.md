# Studio Slides

Single-app Slidev workspace managed with Turborepo.

## Commands

- `bun run dev` starts the Slidev dev server
- `bun run build` builds the deck to `apps/slides/dist`
- `bun run export` exports the deck PDF to `apps/slides/dist/slides.pdf`

## Project Structure

- `apps/slides` contains the Slidev deck
- `turbo.json` defines the workspace tasks
